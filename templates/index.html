FastAPI SAML SSO with Okta

This is a demo application showing SAML Single Sign-On integration with Okta.

Setup Instructions:
  1. Update SAML_SETTINGS in the code with your Okta values
  2. In Okta, set Single sign-on URL to: http://localhost:8000{{ root_path }}/saml/acs
  3. In Okta, set Audience URI to: http://localhost:8000{{ root_path }}/saml/metadata
  4. Click the login button below to test
Login with Okta SSO

Available Endpoints:

  • GET {{ root_path }}/saml/login - Initiate SSO login
  • GET {{ root_path }}/saml/logout - Initiate SSO logout
  • POST {{ root_path }}/saml/acs - Assertion Consumer Service (callback from Okta)
  • GET {{ root_path }}/saml/metadata - Service Provider metadata
  • GET {{ root_path }}/saml/sls - Single Logout Service
  • GET {{ root_path }}/protected - Example protected route
